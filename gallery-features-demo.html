🚀 使用流程演示

相簿瀏覽

訪客首先看到精美的相簿封面，每個相簿都有清晰的分類和照片數量顯示。

// 相簿數據結構 { id: 'portrait', name: '人像攝影', cover: 'cover-image.jpg', description: '捕捉人物的神韻與情感', photoCount: 15 }

進入相簿

點擊相簿卡片後，進入該相簿的照片展示頁面，使用JavaScript精確控制的瀑布流布局。

// 瀑布流布局算法 const shortestColumnIndex = columnHeights.indexOf(Math.min(...columnHeights)); const x = shortestColumnIndex * (columnWidth + gap); const y = columnHeights[shortestColumnIndex];

搜索和篩選

使用搜索框和標籤篩選功能，快速找到特定的照片。支持多重篩選條件組合。

// 篩選邏輯 const filteredPhotos = photos.filter(photo => photo.title.includes(searchQuery) && selectedTags.every(tag => photo.tags.includes(tag)) );
