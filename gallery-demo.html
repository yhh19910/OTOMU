📸 照片畫廊布局比較

❌ 舊版布局 - 固定高度裁剪

所有照片被強制裁剪成相同高度，可能會失去重要的圖片內容

缺點：
  • 照片被裁剪，可能失去重要內容
  • 不尊重攝影師的原始構圖
  • 直向照片受影響最大

✅ 新版布局 - 瀑布流原始比例

保持每張照片的原始比例，創造自然的瀑布流效果

優點：
  • 保持原始比例 - 尊重攝影作品的完整性
  • 瀑布流布局 - 自然美觀的視覺效果
  • 響應式設計 - 適配不同螢幕尺寸
  • 優雅動畫 - 懸停時的微妙互動效果

🔧 技術實現

CSS Columns 布局：

.photo-grid {
  column-count: 3; /* 桌面版3列 */
  column-gap: 1.5rem;
  column-fill: balance;
}

.photo-item img {
  width: 100%;
  height: auto; /* 保持原始比例 */
  break-inside: avoid;
}

響應式斷點：

  • 手機版 (≤768px)：1列
  • 平板版 (769px-1024px)：2列
  • 桌面版 (≥1025px)：3列
