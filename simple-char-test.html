🎉 逐字跳動動畫效果

將鼠標懸停在導航項目上，您會看到每個字符依序跳動的美妙效果！

✨ 效果特色：

  • 波浪式跳動：每個字符間隔0.05秒開始動畫
  • 自然跳躍：兩段式跳動（8px → 4px → 0px）
  • 顏色變化：從橄欖綠漸變到深橄欖綠
  • 流暢過渡：0.3秒的平滑過渡效果

🔧 技術實現：

  • 每個字符包裝在獨立的 <span> 元素中
  • 使用 CSS :nth-child() 選擇器設定不同延遲
  • CSS @keyframes 定義跳動動畫
  • 支援中文、日文、英文和符號
